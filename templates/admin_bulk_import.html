Bulk Import Users

Upload a CSV file to create multiple student accounts at once.

CSV Format

Your CSV must include these columns:

username,email,password,full_name,role
student1,student1@example.com,Pass1234,Student One,Student
student2,student2@example.com,Pass1234,Student Two,Student
teacher1,teacher1@example.com,Pass1234,Teacher One,Teacher
  • username, email, password — required
  • full_name — optional
  • role — optional (defaults to Student; use Student, Teacher, or Admin)
  • Students are auto-approved; Teachers/Admins require manual approval
  • Duplicate usernames or emails are skipped with an error
